 Logistic Regression - Binary Classification

 📌 Objective
Build a binary classifier using Logistic Regression.

 🛠 Tools Used
- Python
- Scikit-learn
- Pandas
- Matplotlib

 📊 Dataset
Breast Cancer Wisconsin Dataset

 ⚙️ Steps Performed
1. Loaded dataset
2. Train-test split
3. Feature scaling
4. Logistic regression model training
5. Evaluation using:
   - Confusion Matrix
   - Precision, Recall
   - ROC-AUC

 📈 Results
Accuracy: 97%
ROC-AUC Score: 0.99

 📚 Conclusion
Logistic Regression performs well for binary classification problems and gives high accuracy on this dataset.
